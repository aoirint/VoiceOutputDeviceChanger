# Changelog

## Unreleased

## v0.1.0-alpha.1 - 2026-07-23 UTC

### Added

- Added a settings option that routes other players' voices to a selected
  Windows audio output device.
- Added temporary routing to the Windows multimedia default when a saved
  endpoint is missing.
- Kept remote voices on the normal game output when alternate routing is
  unavailable.

### Notes

- Compatibility: Lethal Company v81.
    - Steam Build ID: `22825947`
    - Steam Manifest ID: `6423525044216269478`
- Install the mod only on clients that want to change remote voice output.
- This alpha is published on GitHub for testing. Clean-profile multiplayer and
  audio-device runtime validation remains pending.
